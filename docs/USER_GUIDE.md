## 系統概覽

法式編織工作室是一套專為法式編織手作設計的管理系統，涵蓋作品記錄、客戶管理、訂單追蹤、線材庫存與繩長計算等功能。

### 導覽列

- **桌面版**：頂部水平導覽列，點擊各分頁名稱即可切換
- **手機版**：點擊右上角選單圖示開啟側邊抽屜，選擇分頁後自動關閉

## 作品管理

### 瀏覽作品（/works）

- 以卡片方式呈現所有作品
- **篩選功能**：點擊上方標籤可依狀態篩選
  - 全部 / 製作中 / 已完成 / 販售中 / 已售出
- 每張卡片顯示：作品圖片、名稱、客戶、價格、花朵與變化數量

### 新增作品（/works/new）

1. **基本資訊**
   - 作品名稱（必填）
   - 客戶：從下拉選單選擇，或點「+ 新增」建立新客戶
   - 狀態：製作中 / 已完成 / 販售中 / 已售出
   - 販賣價格（NT$）
   - 說明

2. **成品圖片**
   - 點擊上傳，支援多張圖片
   - 圖片會自動上傳至雲端儲存

3. **使用線材**
   - 點「+ 新增線材」加入列
   - 選擇線材 → 輸入長度（cm）→ 輸入數量
   - 找不到線材？點「建立新的線材」直接新增
   - 點「x」移除已選線材

4. **使用編法**
   - 點「+ 新增編法」加入列
   - 選擇編法 → 輸入使用次數 → 備註
   - 系統預設 10 種常見編法（平結、雀頭結、螺旋結等）

5. **作品細節**
   - 花朵數量、變化數量
   - 靈感來源、寓意、特別之處

6. 點「儲存作品」完成建立

### 查看作品（/works/作品ID）

- 完整顯示作品所有資訊
- 圖片以網格排列，可點擊放大
- 右上角「編輯」按鈕可進入編輯頁面

### 編輯作品（/works/作品ID/edit）

- 載入現有資料，所有欄位皆可修改
- **圖片管理**：
  - 現有圖片以縮圖顯示，滑鼠移上去出現「✕」可移除
  - 下方可上傳新圖片，新舊圖片會合併
- 修改完成後點「更新作品」

## 訂單管理（/orders）

### 瀏覽訂單

- 以卡片方式呈現，每張顯示：客戶、狀態、日期、金額、運送方式、品項標籤
- **狀態切換**：每張訂單右上角有下拉選單，可直接切換
  - 待出貨 → 已寄送 → 已送達
  - 或標記為已取消
- 點「詳情」展開查看完整資訊

### 新增訂單

1. 點「+ 新增訂單」
2. **選擇客戶**（必填）：選擇後會自動帶入該客戶的預設運送方式
3. **運送方式**：
   - 宅配 → 輸入地址
   - 超商取貨 → 選擇超商（7-ELEVEN / 全家 / 萊爾富 / OK超商）→ 輸入門市名稱
4. **訂購品項**：
   - 選擇作品 → 系統自動帶入作品價格
   - 可修改價格與數量
   - 點「+ 新增品項」加入更多作品
   - 底部自動計算合計金額
5. 設定狀態與備註
6. 點「建立訂單」

### 訂單回饋紀錄

每張訂單展開後可管理回饋紀錄：

1. 點「查看回饋紀錄」載入該訂單的回饋
2. 點「+ 新增回饋」新增紀錄：
   - 類型：回饋 / 詢問 / 溝通 / 其他
   - 內容（文字）
   - 附加圖片（可多張）
3. 每筆紀錄可刪除

## 客戶管理（/clients）

### 瀏覽客戶

- 卡片式列表，每位客戶顯示：姓名、電話、運送方式、備註
- 點「詳情」展開查看消費紀錄與往來紀錄

### 新增 / 編輯客戶

- **姓名**（必填）
- **電話**
- **運送方式**：
  - 宅配 → 輸入地址
  - 超商取貨 → 選擇超商 + 門市名稱
  - 此為預設運送方式，建立訂單時會自動帶入
- **備註**：例如喜好、介紹人等

### 消費紀錄

展開客戶卡片可查看該客戶所有作品：
- 顯示作品縮圖、名稱、狀態、價格、日期
- 點擊可跳轉至作品詳情

### 往來紀錄

展開客戶卡片可管理互動紀錄：
1. 點「+ 新增紀錄」
2. 選擇類型：回饋 / 詢問 / 溝通 / 其他
3. 輸入內容，可附加圖片
4. 此處顯示該客戶的所有紀錄（含訂單綁定與非訂單的）

## 線材管理（/threads）

### 瀏覽線材

表格式列表，顯示：顏色預覽、名稱、材質、粗細、來源

### 新增線材

1. 點「+ 新增線材」
2. 填寫：
   - 顏色名稱（必填）：例如「酒紅色」
   - 顏色：用色彩選取器選擇
   - 材質：例如蠟線、棉線、尼龍
   - 粗細（mm）
   - 來源：例如蝦皮、手藝材料行

### 編輯 / 刪除

- 每列右側有「編輯」和「刪除」按鈕
- 刪除線材會影響已關聯的作品

### CSV 批次匯入

1. 點「CSV 匯入」按鈕
2. 點「下載 CSV 範例檔案」取得格式模板
3. CSV 格式：

   ```
   color_name,color_hex,material,thickness_mm,source
   酒紅色,#8B0000,蠟線,0.8,蝦皮
   天藍色,#87CEEB,棉線,1.0,手藝材料行
   金色,#FFD700,尼龍,0.5,
   ```

   - `color_name`：必填
   - `color_hex`：選填（預設 #000000）
   - `material`、`thickness_mm`、`source`：選填

4. 選擇 CSV 檔案 → 點「開始匯入」
5. 匯入結果會即時顯示

## 繩長計算機（/calculator）

用於計算編織作品所需的繩長。

### 使用方式

1. **選擇結型**：從預設清單選擇（平結、螺旋結、雀頭結等），系統自動帶入對應倍率
2. **成品長度（cm）**：輸入希望的完成長度
3. **繩數**：輸入使用的繩子數量
4. **自訂倍率**（選填）：如果預設倍率不適合，可手動覆蓋

### 計算結果

- **每條繩長** = 成品長度 × 倍率
- **總繩長** = 每條繩長 × 繩數
- 結果同時顯示 cm 與 m 單位

### 預設結型倍率

| 結型 | 倍率 |
|------|------|
| 平結 | 4.0x |
| 螺旋結 | 5.0x |
| 雀頭結 | 3.5x |
| 蛇結 | 6.0x |
| 金剛結 | 5.5x |
| 中國結 | 7.0x |
| 纏繞編 | 3.0x |

## 常見問題

**Q: 圖片上傳失敗？**
請確認圖片格式為 JPG、PNG 等常見格式，單張不超過 50MB。

**Q: CSV 匯入出現亂碼？**
請確保 CSV 檔案以 UTF-8 編碼儲存。如果使用 Excel，另存新檔時選擇「CSV UTF-8」格式。

**Q: 刪除客戶會怎樣？**
該客戶的相關作品中，客戶欄位會被清除，但作品本身不會被刪除。

**Q: 訂單建立後還能修改嗎？**
目前可以透過下拉選單修改訂單狀態。如需修改其他欄位，請聯繫管理員。
